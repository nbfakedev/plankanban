# ADR-0001: LLM Gateway на сервере

## Контекст
Нужен выбор провайдера/модели из UI, но ключи нельзя держать в браузере.

## Решение
Ввести серверный LLM Gateway (`/api/llm/chat`) с адаптерами провайдеров.
Ключи — только на сервере (env/DB), UI хранит только provider/model.

## Последствия
+ можно добавлять провайдеров без правок фронта
+ единая точка лимитов/аудита/бюджета
- нужен backend и дисциплина по секретам
